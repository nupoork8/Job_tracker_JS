# Job Application Tracker

A simple Job Application Tracker built using Vanilla JavaScript, HTML, and CSS. The application helps users keep track of job applications by storing company details, role information, and application status in an interactive dashboard.

## Features

* Add new job applications
* Track application status (Applied, Interview, Rejected)
* Color-coded status indicators
* Delete applications
* Form validation for required fields
* Dynamic UI updates using DOM manipulation
* Responsive and clean user interface

## How It Works

1. Enter the company name.
2. Enter the role applied for.
3. Select the application status.
4. Click "Add Application".
5. The application is displayed as a card.
6. Status is visually highlighted using color indicators.
7. Applications can be removed using the Delete button.
